# Rice_Image_CNN_Classifier
Classifies images of rice into 5 labes, each being a different kind of rice. 

CNN initially trained and reached training accuracy of .988 with a validation loss of .14. Rice image dataset takken from kaggle. 
Dataset is loaded from kaggle by importing kagglehub and the images are loaded into training, validation, and test sets at a 70/15/15 percent split.
